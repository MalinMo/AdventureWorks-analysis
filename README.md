# AdventureWorks – Försäljningsanalys

Detta är en **individuell inlämningsuppgift** där försäljningsdata från databasen AdventureWorks analyseras med SQL och Python.

Analysen omfattar grundläggande försäljningsanalyser samt en djupanalys med fokus på regional försäljningsoptimering.

## Teknik och miljö
- Python 3.13.7  
- Pandas  
- Matplotlib  
- NumPy  
- SQL (AdventureWorks-databasen)

Analysen körs i Jupyter Notebook och visualiseringar skapas med Matplotlib.
